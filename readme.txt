== Description ==

Integrates your latest tweets in your weblog.

**Features**
    
    *  Simple (no configuration required)
    *  Customizable
    *  Fast


**Usage**

Either add the provided widget to the sidebar or use the following code:

`<?php twitter_messages("username"); ?>`



== Installation ==

Extract the zip file into the directory /wp-content/plugins/ and activate the plugin in the admin section of your weblog.
